# Remote Steering Wheel
This project is designed for car racing gamers who has no steering wheel at home, but wishing to steer their phone as an alternative.

You can build the project using [Visual Studio Code](https://code.visualstudio.com/) or using javac

## Instruction for general users:
 1. Install java 17+.
 2. Download RemoteSteeringWheel-xxx.jar from [Latest Release](https://github.com/KrisanaponMeksamoot/RemoteSteeringWheel/releases/latest).
 3. Run `java -jar RemoteSteeringWheel-xxx.jar` and start racing your car.
 4. Connect your mobile device and your computer to the same network.
 5. Open https://\<computer ip\>:8443/ on your mobile device.

### Note:
You have to connect your mobile device to the same *local* network as your computer.
The more direct, the better.

You can obtain your computer ip address from
- network setting
- \> `ipconfig`
- $ `ip address`
